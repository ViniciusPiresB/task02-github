# Rick And Morty Web App

Este repositório contém um projeto de aplicação web que consome a [API do Rick and Morty](https://rickandmortyapi.com/api/character) para exibir informações sobre os personagens da série. O projeto visa criar uma interface amigável para explorar os dados fornecidos pela API.

## Instruções da Prática

O grupo deverá:
- Criar o repo público – task02-grupo06
- Readme:
  - Adicionar conteúdos de filmes ou séries, ou música, ou bandas, ou livro 
  - Imagens
  - Links para sites
  - Nomes dos integrantes do grupo
- Abaixo: tópicos da documentação que vocês escolheram
- feat e commit em inglês ou português 
- Quais os integrantes que trabalham no readme e quais as branches
- O repositório precisa ter um HTML com algum conteúdo relacionado ao escolhido.
  - Imagens
  - Estilização (não é obrigatório para grupos que não tenham expertise nesta parte)
- Flow do Git: feat → develop → main;
- Todos deverão commitar, code review e participar de PRs
